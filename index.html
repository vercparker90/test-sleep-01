MacBook Sleep Wake Issue: Black Screen After Sleep — Simple Fix

One of the most common complaints among MacBook users recently is a serious issue with waking the device from sleep. After closing the lid and reopening it later, the screen remains completely black, even though the keyboard appears to respond. This problem started appearing with macOS Sequoia and has not been fully resolved even in newer updates.

Step 1: Open Terminal
  • Open Terminal on your Mac — you can find it via Spotlight (⌘ + Space) by typing "Terminal"
    •  Step 2: Run the Fix Command
  • Paste the command below into Terminal and press Enter
    •
  • Enter your Mac password if prompted, then restart your Mac
    •
    Terminal
    sudo pmset -a standby 0 && sudo dscacheutil -flushcache && sudo killall -HUP mDNSResponder && sudo purge
    Copy
    What this fix does:
  • Disables standby mode — a deep sleep state macOS enters after extended inactivity
    •
  • Keeps your Mac in a lighter sleep state so it wakes more reliably without triggering the black screen
    •
  • Does not affect your personal files, documents, or application settings
    •  What causes the issue

    The issue appears to be related to how macOS handles power states and sleep transitions. In some cases, the graphics subsystem or background services fail to properly reinitialize after waking, resulting in a non-responsive display while the system itself is still partially active. Thousands of users across Apple forums have reported the same behavior since macOS Sequoia.

    Affected Mac models
  • MacBook Pro — all models running macOS Sequoia or later
    •
  • MacBook Air — M1, M2, and M3 chips confirmed affected
    •
  • Mac mini — users report the same black screen behavior after sleep
    •
  • iMac — external display users may also experience display reinitalization failures
    •
  • Mac Studio / Mac Pro — less common but the same workaround applies
    •  Is it safe?

    Yes. The command only changes a power management setting — it does not delete files or modify system data. Disabling standby simply prevents macOS from entering the deepest sleep state, keeping your Mac in a lighter idle mode instead. macOS will continue to function normally, and you can reverse the change at any time by running the same command with a value of 1.

    Will this affect battery life?

    There may be a minor impact on battery when the lid is closed for extended periods, since standby mode is designed to reduce power consumption during long idle sessions. However, for most users the difference is negligible — especially compared to the inconvenience of a forced reboot every time the Mac wakes from sleep. If battery life is a concern, you can re-enable standby with: sudo pmset -a standby 1.

    Additional tips if the issue persists
  • Reset the SMC (System Management Controller) — this can resolve power state issues on Intel Macs
    •
  • Check System Settings → Battery → Options and disable Enable Power Nap to reduce background wake activity
    •
  • Keep macOS up to date — Apple continues releasing patches that may include a permanent fix
    •
  • If the black screen persists, try pressing Ctrl + Shift + Eject (or Power) to put only the display to sleep and wake it separately
    •
  • Report the issue to Apple via Feedback Assistant to help accelerate an official fix
